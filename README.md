# bigharshrag.github.io

Personal site, built with [Astro](https://astro.build).

## Develop

```sh
npm install
npm run dev      # local preview at http://localhost:4321
npm run build    # production build into dist/
```

## Publish

Push to `master`. GitHub Actions (`.github/workflows/deploy.yml`) builds with Node 20 and deploys `dist/` to GitHub Pages — no manual step.
